# kokkoro-plugin-guild

> 我不想打公会战

## 安装

``` shell
# 切换至 bot 目录
cd bot

# 安装 npm 包
npm i kokkoro-plugin-guild
```

在 [kokkoro](https://github.com/kokkorojs/kokkoro) 成功运行并登录后，发送 `>enable guild` 即可启用插件  
首次在当前群聊使用需要输入 `设置国服公会` 初始化游戏服务器
